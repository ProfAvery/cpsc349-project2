This project was made by Yu Li, Roger Hanna, David Bonnaud, and Michael Nguyen. We used the Unsplashed dataset to build a site consisting of pet photos from various photographers. Some of the metadata such as the camera used and its settings along with city, location, etc. on each photo were extracted so more info is displayed with each photo. The dataset we used was from the Unsplash dataset.
